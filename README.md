# OnGas-Server
Projeto que surgiu da necessidade de tornar encomendas de gás de cozinha para residências algo mais prático através de um aplicativo.

## Destinatários
- Distribuidoras de gás de cozinha
- Donos de casas

## Ferramentas
- Hibernate
- Mysql
- Gson
- Jersey
